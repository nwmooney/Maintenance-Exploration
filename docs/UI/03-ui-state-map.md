# CMX UI Surfaces

This document defines the primary UI surfaces for CMX.
Structure is inspired by Samsara Maintenance (asset-centric, action-forward).

---

## Core Surfaces

### 1. Maintenance Inbox
- Unified queue:
  - Defects
  - PM due
  - Invoice exceptions
  - Approval required
- Sorted by urgency

---

### 2. Asset Overview
- Asset health
- Open issues
- Active work orders
- Upcoming PM

---

### 3. Work Order Detail
- Event timeline
- Status banner
- Contextual actions

---

### 4. Documents & Invoices
- Upload
- Validation
- Exception handling

---

### 5. Insights (Future)
- Failure trends
- PM compliance
- Vendor performance
