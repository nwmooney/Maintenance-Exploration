# CMX A2UI Partner SDK – Integration Guide

## Purpose

This document describes how external partners integrate with **CMX (Maintenance)** using the
**A2UI Runtime Bundle** published by J.B. Hunt.

Partners do **not** generate UI.
Partners **render deterministic UI surfaces** authored and governed by CMX.

---

## What CMX Publishes

CMX publishes a versioned A2UI runtime bundle that includes:

- Approved A2UI catalog
- Event → UI payload mappings
- Fallback UI for unknown events
